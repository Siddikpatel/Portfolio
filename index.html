About Me

I am currently pursuing a Master's in Applied Computer Science at Dalhousie University, Halifax and expect to graduate in Winter 2026. Alongside my studies, I am working on projects that integrate technology with real-world applications.

One of my projects is a mobile application built with React Native that allows users to upload images of waste materials. The application identifies the type of material in the image and provides guidance on proper disposal, recycling, or potential reuse. To encourage sustainable behavior, the app includes a leaderboard system, where users earn points based on their environmentally conscious actions.

My other project is a cloud-based web application, designed as a tier-3 architecture running entirely on AWS. This project leverages various AWS services, including EC2 for processing, S3 for storage, DynamoDB for metadata management, API Gateway for communication, and Lambda for event-driven tasks. The goal is to build a scalable and efficient system that fully utilizes cloud-native technologies.

I am currently looking for an internship opportunity starting in Fall 2025, where I can apply my skills in software development, cloud computing, and full-stack development while gaining industry experience.
